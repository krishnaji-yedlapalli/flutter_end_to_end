# Flutter End to End

A Flutter project to learn how to build the app on a Multi-platform using a single code base.


# Currently Available Features :
1. Theme
2. Routing
3. Localization
4. Offline support using the Sqflite plugin
5. Bloc Pattern
6. Deep Linking
7. Push Notification Local and Remote
